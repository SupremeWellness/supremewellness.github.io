
SUPREME WELLNESS WEBSITE SETUP

To change your practitioner login password:
1. Open the file 'config.js' using Notepad or any text editor.
2. Change the line:
   const ADMIN_PASSWORD = "changeme";
   to:
   const ADMIN_PASSWORD = "yournewpassword";
3. Save the file and upload it back to your hosting.

Your Cash App handle is currently set as: $SupremeWellness
To change it, update it directly in the HTML or let your developer adjust it for dynamic use.
